# findings/ — Research findings notes

Each file records the results of one completed experiment.
The research loop produces findings notes via the analyze node.

**Naming convention:** `find-<id>.md`

**Required frontmatter:**
```yaml
---
type: findings
citekey: find-<id>
title: <Finding title>
description: <one-sentence summary of the note (optional; WARN if empty)>
---
```
