# concepts/ — Distilled concept notes

Each file captures a durable conceptual claim that emerged from the literature synthesis.
The synthesize node soft-couples to this directory (critic-judged, not file-gated).

**Naming convention:** `<concept-slug>.md`

**Required frontmatter:**
```yaml
---
type: concepts
title: <Concept title>
description: <one-sentence summary of the note (optional; WARN if empty)>
---
```
